This is a demonstration of anchors that change position dependent on the location of the other element in a connection. Each connection is decorated with a jsPlumb.Overlays.Diamond, located halfway along the connector.

This demonstration uses jsPlumb 1.3.3, jQuery 1.6.0 and jQuery UI 1.8.13.
