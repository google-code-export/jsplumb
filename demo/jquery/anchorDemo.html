This page provides a demonstration of how to work with Anchors.

All the green windows are anchored to Window 1 using custom anchor positions on Window 1, except Window 4, which uses the BottomCenter anchor provided by jsPlumb.

Additionally, this page shows the usage of a different endpoint for each end of the connector, and a gradient in the connector. If you are using IE<9, you will see green connectors with no gradient. VML, which jsPlumb uses for older IE, does not support stroking gradients.

This demonstration uses jsPlumb 1.3.3, jQuery 1.6.0 and jQuery UI 1.8.13.
